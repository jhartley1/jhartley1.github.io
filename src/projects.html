Locurity

I joined as the first engineer and built a platform for 2FA based on 'ambient authentication'. User approvals for authentication built an accompanying signature of their local environment that could be compared on the machine initiating login, eventually no longer requiring explicit confirmation. We believed we could use the phone's gyroscope to identify a user and that we could compare signals in the environment like WiFi, Bluetooth, and temperature across the initiating machine and the user's phone.

In practice, our system used a mobile application as the second factor and we would automatically approve 2FA requests when the mobile device was in a known location. Known locations were accumulated through previous approvals in the same place using a clustering algorithm.
